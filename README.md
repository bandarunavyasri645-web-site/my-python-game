# Guess Number Game

This is a simple web-based guessing game built with HTML, CSS, and JavaScript.

## How to play
Open `index.html` in your web browser.

1.  Enter a number between 1 and 10.
2.  Click "Guess" or press Enter.
3.  The game will tell you if your guess is too high, too low, or correct.
4.  Once you guess correctly, you can click "Play Again" to start a new game.
